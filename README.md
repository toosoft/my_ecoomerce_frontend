# My Ecommerce Frontend

A modern, responsive eCommerce frontend built with **Next.js**, featuring product browsing, cart management, authentication, and seamless API integration with a Spring Boot backend.

---

## Features

- Add to cart (guest & authenticated users)
- Cart sync with backend
- Authentication (login/register)
- Product listing & details page
- Optimistic UI updates (instant feedback)
- Responsive UI with Tailwind CSS
- Global state management using Context API

---

## Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **State Management:** React Context API
- **Backend API:** Spring Boot (REST API)
